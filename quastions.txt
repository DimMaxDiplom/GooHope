Бекенд:
    [auth] Сервис авторизации:
        - [register] регистрация (имя, почта, пароль) -> генерация кода, который приходит на email,
                                              для подтверждения почты
        - [email_confirmation] подтверждение почты (почта, код) -> завершение регистрации, генерация токена
        - [login] логин (почта, пароль) -> получения токена, для авторизации в остальных сервисах
        - [reset_password] восстановление пароля (почта) -> отправку на почту код сброса пароля
        - [change_password] смена пароля (почта, старый пароль)
        База данных:
            PostgreSQL:
                Users (id, name, email, password_hash, register_date)
            Redis (Что бы сохранять на 10 минут код для подтверждения почты)
 
    [payments] Сервис оплаты:
        - покупка игры (id игры) -> добавления игры в библиотеку
        - возврат игры (id игры) -> возврат средств за игру, удаление игры из библиотеки
        База данных:
            PostgreSQL (храним историю транзакций):
 
    [library] Сервис библиотеки:
        - получение списка купленных игр (id юзера) -> список игр
        - загрузки игры из [apps_storage]
    [apps_storage] Сервис хостинга файлов игр:
        - Отправка файлов игры десктопному клиенту по SFTP
    [search] Сервис поиска:
        - Поиск игры по описанию
    [socials] Сервис социальных штук (друзья, сообщения, комментарии):

Браузерный клиент:
Страница магазига:
    - список новых игр
    - список популярных
    - список рекомендуемых игр
    - список игр по скидке
    - список недавних купленных игр друзей (???)
 
На странице регистрации:
    - Форма с 4 полями (имя, почта, пароль, повторение пароля) и кнопка регистрации
                    -> отправка запроса на [auth.register],
                    -> перенаправление на страницу подтвержедния почты
 
На странице подтвержедния почты:
    - Форма с 1 полем (код) -> [auth.email_confirmation] -> получение токена
 
На странице авторизации:
    - Форма с 2 полями (почта, пароль) -> [auth.login]
    - Кнопка восстановления пароля
